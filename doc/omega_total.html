Berechnung der Winkelgeschwindigkeit ω(φ) aus der Gesamtkraft

1. Physikalische Grundlagen

Die Winkelgeschwindigkeit ω(φ) kann direkt aus der wirkenden Gesamtkraft F berechnet werden, indem der Drehimpulserhaltungssatz mit der Tangentialkomponente der Kraft verknüpft wird.

2. Mathematische Herleitung

Schritt 1: Drehimpulserhaltung

Der spezifische Drehimpuls h (pro Masse) ist konstant:

\[ h = r^2 \omega(\phi) = \text{konstant} \]

Ableitung nach der Zeit liefert:

\[ \frac{dh}{dt} = 2r \dot{r} \omega + r^2 \dot{\omega} = 0 \] \[ \Rightarrow \dot{\omega} = -\frac{2 \dot{r}}{r} \omega \]

Schritt 2: Tangentialkraft

Die Tangentialkomponente der Gesamtkraft F berechnet sich als:

\[ F_\phi = \vec{F} \cdot \hat{\phi} = -F_x \sin\phi + F_y \cos\phi \]

Die Änderung des Drehimpulses ist:

\[ \dot{h} = r F_\phi \]

Schritt 3: Differentialgleichung für ω(φ)

Kombination der Ergebnisse führt zur Differentialgleichung:

\[ \frac{d\omega}{d\phi} = -\frac{2 v_r}{r} + \frac{F_\phi}{r \omega} \]

wobei \( v_r = \dot{r} \) die Radialgeschwindigkeit ist.

3. Lösung der Differentialgleichung

Allgemeine Lösung

Die Winkelgeschwindigkeit ergibt sich aus der Integration:

\[ \omega(\phi) = \omega_0 \exp\left(-\int \frac{2 v_r}{r} d\phi\right) + \int \frac{F_\phi}{r \omega} d\phi \]

In der Praxis wird dies numerisch gelöst.

Numerische Integration

Mit dem Euler-Verfahren für einen kleinen Winkel Δφ:

\[ \omega_{n+1} = \omega_n + \Delta\phi \left( -\frac{2 v_r}{r} + \frac{F_\phi}{r \omega_n} \right) \]

4. Anwendung auf Planetenbahnen

Berechnungsschritte

  1. Berechne die Gesamtkraft F auf den Körper
  2. Bestimme die Tangentialkomponente Fφ
  3. Berechne die Radialgeschwindigkeit vr
  4. Integriere die DGL für ω(φ)

Wichtiger Hinweis

Diese Methode berücksichtigt automatisch alle Störkräfte durch andere Körper sowie relativistische Effekte, da sie direkt auf der Gesamtkraft F basiert.

5. Beispiel: Kreisbahn (Fφ = 0)

Für eine ungestörte Kreisbahn (vr = 0, Fφ = 0) ergibt sich:

\[ \frac{d\omega}{d\phi} = 0 \]

⇒ ω = konstant, wie erwartet für eine Keplerbahn.
