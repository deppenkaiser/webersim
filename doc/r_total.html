Vektorielle Bahngleichung r⃗(φ) mit Weber-Gesamtkraft

1. Vektordefinitionen

Polarkoordinatenbasis:

\[ \begin{align*} \hat{r} &= \cos\phi\,\hat{x} + \sin\phi\,\hat{y} \\ \hat{\phi} &= -\sin\phi\,\hat{x} + \cos\phi\,\hat{y} \end{align*} \]

Ortsvektor:

\[ \vec{r}(\phi) = r(\phi)\hat{r} \]

2. Bewegungsgleichung mit Weber-Kraft

2.1 Vektorielle Beschleunigung

\[ \vec{a} = (\ddot{r} - r\dot{\phi}^2)\hat{r} + (r\ddot{\phi} + 2\dot{r}\dot{\phi})\hat{\phi} \]

2.2 Weber-Gesamtkraft (vektoriell)

\[ \vec{F}_{\text{Weber}} = -\frac{GMm}{r^2}\left(1 - \frac{\dot{r}^2}{c^2} + \frac{r\ddot{r}}{2c^2}\right)\hat{r} \]

Für Mehrkörpersysteme:

\[ \vec{F}_{\text{Gesamt}} = \sum_j -\frac{GM_j m}{r_j^2}\left(1 - \frac{\dot{r}_j^2}{c^2} + \frac{r_j\ddot{r}_j}{2c^2}\right)\hat{r}_j \]

3. Exakte Differentialgleichung

3.1 Radialkomponente

\[ \ddot{r} - r\dot{\phi}^2 = -\frac{GM}{r^2}\left(1 - \frac{\dot{r}^2}{c^2} + \frac{r\ddot{r}}{2c^2}\right) \]

3.2 Azimutalkomponente

\[ r\ddot{\phi} + 2\dot{r}\dot{\phi} = 0 \quad \Rightarrow \quad h = r^2\dot{\phi} = \text{konst.} \]

4. Lösung der Bahngleichung

4.1 Substitution

\[ u = \frac{1}{r}, \quad \dot{r} = -h\frac{du}{d\phi}, \quad \ddot{r} = -h^2 u^2 \frac{d^2u}{d\phi^2} \]

4.2 Transformierte Gleichung

\[ \frac{d^2u}{d\phi^2} + u = \frac{GM}{h^2} + \frac{3GM}{c^2}u^2 - \frac{GM}{2c^2 h^2}\left(\frac{du}{d\phi}\right)^2 \]

4.3 Exakte Lösung (bis zur 1. Ordnung in 1/c²)

\[ r(\phi) = \frac{a(1-e^2)}{1 + e\cos(\kappa\phi)} \left[1 + \frac{3G^2M^2}{c^2h^4}\left(1 + \frac{e^2}{2} + e\phi\sin(\kappa\phi)\right)\right] \]

mit \(\kappa = \sqrt{1 - \frac{6GM}{c^2 a(1-e^2)}}\)

5. Vektorielle Darstellung

5.1 Vollständiger Ortsvektor

\[ \vec{r}(\phi) = r(\phi)\hat{r} = \frac{a(1-e^2)}{1 + e\cos(\kappa\phi)} \left[1 + \mathcal{O}(1/c^2)\right] (\cos\phi\,\hat{x} + \sin\phi\,\hat{y}) \]

5.2 Geschwindigkeitsvektor

\[ \vec{v}(\phi) = \dot{r}\hat{r} + r\dot{\phi}\hat{\phi} = h\left(-\frac{du}{d\phi}\hat{r} + u\hat{\phi}\right) \]

6. Physikalische Interpretation

Term Bedeutung
\(\kappa\phi\) Relativistische Winkelkorrektur (Periheldrehung)
\(e\phi\sin(\kappa\phi)\) Säkulare Störung der Bahn
\(\mathcal{O}(1/c^2)\) Höhere relativistische Korrekturen

Die vektorielle Darstellung zeigt explizit:

  • Radiale und azimutale Komponenten bleiben getrennt
  • Die Weber-Kraft wirkt nur in \(\hat{r}\)-Richtung
  • Drehimpulserhaltung erzwingt die Form der \(\hat{\phi}\)-Komponente
