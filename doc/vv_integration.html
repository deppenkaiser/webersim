N-Körper-Integration mit Velocity-Verlet

1. Physikalische Grundgleichungen

Gravitationskraft zwischen Körper i und j:
Fij = -G mimj (xi - xj) / |xi - xj|3

2. Velocity-Verlet Algorithmus

Initialisierung (t = 0)

  1. Setze Startpositionen xi(0) und Geschwindigkeiten vi(0)
  2. Berechne Anfangsbeschleunigungen:
    ai(0) = (1/mi) ∑j≠i Fij(0)

Zeitschritt t → t + Δt

  1. Halber Geschwindigkeitsschritt:
    vi(t + Δt/2) = vi(t) + ½ ai(t) Δt
  2. Positionsupdate:
    xi(t + Δt) = xi(t) + vi(t + Δt/2) Δt
  3. Neue Beschleunigungen:
    ai(t + Δt) = (1/mi) ∑j≠i Fij(t + Δt)
  4. Vollständiger Geschwindigkeitsschritt:
    vi(t + Δt) = vi(t + Δt/2) + ½ ai(t + Δt) Δt

3. Energieerhaltung

Eges = ∑i ½ mi|vi|2 - G ∑i<j (mimj / |xi - xj|)

4. Zeitschrittkontrolle

Optimal: Δt ≈ T/104 (T = typische Umlaufzeit)
